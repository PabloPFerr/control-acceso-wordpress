Sistema de control de acceso y registro de asistencia para empleados en WordPress.

== Description ==

WP Control Acceso es un plugin de WordPress que proporciona un sistema completo para gestionar el control de acceso y registro de asistencia de empleados. 

Características principales:

* Registro de entrada y salida de empleados
* Panel de control para administradores
* Reportes personalizados de asistencia
* Sistema de roles y permisos integrado
* Interfaz intuitiva y fácil de usar
* Exportación de reportes

= Para Empleados =

* Registro fácil de entrada y salida
* Visualización de sus propios reportes de asistencia
* Interfaz simple y directa

= Para Administradores =

* Panel de control completo
* Gestión de empleados
* Visualización de todos los registros
* Generación y exportación de reportes
* Control de permisos y roles

== Installation ==

1. Sube los archivos del plugin al directorio `/wp-content/plugins/wp-control-acceso/`
2. Activa el plugin a través del menú 'Plugins' en WordPress
3. Configura los ajustes básicos desde el panel de administración
4. Crea las páginas necesarias usando los shortcodes proporcionados

== Frequently Asked Questions ==

= ¿Cómo registro la entrada de un empleado? =

Utiliza el shortcode [control_acceso_registro] en cualquier página para mostrar el formulario de registro.

= ¿Cómo pueden los empleados ver sus reportes? =

Usa el shortcode [control_acceso_mis_reportes] en una página para que los empleados puedan ver sus propios reportes.

= ¿Qué roles incluye el plugin? =

El plugin incluye roles para empleados y administradores, cada uno con sus respectivos permisos.

== Privacy Policy ==

Este plugin recopila datos de asistencia de los empleados, incluyendo:
* Hora de entrada y salida
* Nombre de usuario
* ID de empleado

Estos datos son almacenados en la base de datos de WordPress y solo son accesibles por los administradores del sistema y el propio empleado (sus propios datos).
